# Cooler: A COOL Language Compiler in Go


> A compiler for the Classroom Object-Oriented Language (COOL) written in Go

## About COOL

COOL is a small language designed for use in an undergraduate compiler course project. While small enough to be implemented in a few weeks, COOL still has many of the features of modern programming languages, including objects, types, and automatic memory management.

  - Multi-line nested comments `(* ... *)`
